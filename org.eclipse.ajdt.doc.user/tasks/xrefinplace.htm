Using the quick Cross References view

To bring up the quick Cross References view select the element for which you want to show the cross references in the editor and either:

  • Press Alt+Shift+P or
  • Select Navigate > Quick Cross References

To resize the quick Cross Reference view either

  • Grab an edge of the quick Cross References view and resize or
  • Select Resize in the drop down menu on the quick Cross References view and resize when prompted

To move the quick Cross Reference view

  • Select Move in the drop down menu on the quick Cross References view and move when prompted

To persist the position and size of the quick Cross Reference view

  • Select Remember size and location in the drop down menu on the quick Cross References view

To show/hide the Cross References for an entire file

  1. Press Alt+Shift+P to show the Cross References for the current selection in the editor
  2. Press Alt+Shift+P again to show the Cross References for the entire file
  3. Press Alt+Shift+P a third time to return to showing the Cross References for the current selection in the editor

To navigate from the quick Cross References view to the corresponding place in the editor either

  • Select the required entry in the quick Cross References view and press Return on your keyboard or
  • Double click on the required entry in the quick Cross References view

To text filter the contents of the quick Cross References view start typing in the text filter bar at the top of the quick Cross References view and the contents of the view will update to only show contents that start with the letters you have typed.
