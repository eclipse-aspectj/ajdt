Opening an editor on a type

You can open an editor on any type in the workbench. As of AJDT 1.6.2, Aspects appear in the standard Java open type dialog along with classes, interfaces, and enums.

To activate the open type dialog, press Alt+Shift+T or, select Navigate > Open AspectJ Type from the menu bar.
