Clean Install

AJDT is installed using the Eclipse Update Manager. We recommend you use Eclipse 3.0, but AJDT v 1.1.4 will work with Eclipse 2.1.x.

 

1. If you need to use a proxy server to access the internet, the first thing to do is configure the proxy details so that the update manager can contact the AJDT Update Site. From the Window menu select preferences, and then the Install/Update tab.

 

 

2. Create an update site bookmark for the AJDT update site, and start the install procedure.

 

  • Eclipse 3.0
  • Eclipse 2.1

 

In Eclipse 3.0: From the help menu, select Software Updates -> Find and Install… . Select “Search for new features to install” and click next. Click “Add Update Site.”  Enter the name “AJDT update site” and the URL “http://download.eclipse.org/technology/ajdt/30/update”.  Click ok. 

 

For Eclipse 3 users there are two versions of AJDT available. 1.1.12 is the latest stable release of AJDT. 1.2.0 is being released as a series of milestone builds leading up to the final release. Either 1.1.12 or the latest 1.2.0 milestone build can be installed. 1.2.0 contains many enhancements over 1.1.12, but as such is considered to be less stable.

 

Fully expand the AJDT Update Site node that appears, and select “AspectJ.” Click next. Select “Eclipse AspectJ Development Tools 1.1.12” to install 1.1.12, or “Eclipse AspectJ Development Tools 1.2.0.2004..” to install the latest 1.2.0 milestone build.

 

 

Select Next, accept the license agreement, and proceed to the installation.

 

In Eclipse 2.1:

From the help menu select Software Updates -> Update Manager. The Feature Updates view appears in the bottom right-hand corner. 

 

 

Right-click on the background of this view to bring up the context menu and select New -> Site Bookmark. Enter the name “AJDT update site” and the URL “http://download.eclipse.org/technology/ajdt/update”.  Click finish.

 

 

Fully expand the AJDT Update Site node that appears, and select “Eclipse AspectJ Development Tools 1.1.4.”  In the preview pane, select “Install…”

Updating an Existing Installation

 

Proceed as for a clean install, except that the AJDT Update Site bookmark should already exist – so all you need to do is expand it and go. If the version you have installed is the same as the version on the update site (or more recent even), then you will not be presented with any install options.

 

Is Everything OK?

When Eclipse has restarted, selecting Help->About Eclipse Platform should show the AspectJ feature icon alongside the Eclipse one.

 

 

In Eclipse 3.0 the AJDT welcome pages have been integrated into the Eclipse Platform welcome pages. You can view these by selecting "Welcome" from the help menu and then following the Overview, Tutorials, Samples or the What's New links. 

 

 

In Eclipse 2.1.x if it’s not already showing, you can see the welcome page for AJDT by selecting “Welcome…” from the help menu and then “AspectJ Development Tools.”  (It’s a good idea to switch to the Java Perspective at this point if you’re not already in it).

 

 

The blue links in the welcome page are “live” and will initiate the appropriate actions directly in Eclipse.
