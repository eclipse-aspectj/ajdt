Navigating with the Visualiser

To navigate from the affected places in your code shown in the Visualiser view, to the corresponding line of code in the editor, double click on the stripe in the Visualiser view. This opens the editor in the area at the bottom of the Aspect Visualization perspective with the corresponding line of code highlighted.

To open a class or aspect from the Visualiser view, just double click on the required bar.

You can use either your mouse or keyboard to navigate around the Visualiser view itself.
