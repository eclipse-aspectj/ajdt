Using the Generate AJdoc wizard

  1. In the workspace select the project for which you want to generate AJdoc documentation.
  2. From the menu bar select Project > Generate AJdoc...
  3. Select types for AJdoc generation:
    • In the tree control, select the elements for which you want to generate AJdoc.
    • Select the visibility using the radio buttons listed under Create AJdoc for members with visibility
    • Leave the Use Standard Doclet radio button selected
    • Specify the location for the generated AJdoc using the Destination field.
    • Press Finish to create generate AJdoc for the elements you selected or press Next to specify more options.
  4. The AJdoc wizard allows the same options and arguments as the Javadoc wizard. Please see the Java development user guide for more information about Javadoc options.
