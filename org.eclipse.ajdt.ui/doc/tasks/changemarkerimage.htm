Changing the image for advice markers

Images can be configured on a per-aspect basis.

To change the image for advice markers relating to an aspect:

  1. Select the aspect, file or project in the Package Explorer
  2. Right click and select AspectJ Tools > Configure Advice Markers... from the context menu.
  3. Select the aspect that you want to change the marker image for and click Edit....
  4. Select a sample image from the list OR click Browse... and select an image file from the project.
  5. Click OK.
  6. Click OK to apply your changes.
